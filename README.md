# Pandas Data Cleaning Task (Task 5)

## Objective
To clean and preprocess a real-world dataset using Python Pandas and demonstrate data handling skills beyond Excel.

## Dataset
Titanic Dataset (CSV)

## Tools Used
- Google Colab
- Python
- Pandas
- NumPy

## Cleaning Steps Performed
- Loaded dataset using pandas
- Inspected structure and missing values using info() and isnull()
- Filled missing numeric values using median
- Filled missing categorical values using mode
- Removed duplicate records
- Converted datatypes for accurate analysis
- Created a new feature (Age_Group)
- Exported cleaned dataset to CSV

## Deliverables
- Task5_Cleaning.ipynb
- cleaned_data.csv

## Outcome
Developed hands-on experience in data cleaning, transformation, and reproducible analysis using Python.
